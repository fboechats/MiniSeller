# Mini Seller

This is a polished implementation of the Mini Seller challenge.

## What's included
- React + Vite + TypeScript + Tailwind project
- Basic test suite using Vitest + Testing Library

## Run locally

1. Install dependencies:
```bash
npm install
```

2. Run dev server:
```bash
npm run dev
```

3. Run tests:
```bash
npm run test
```
